
# Python tutorial
Read the documentation at <https://pranabdas.github.io/python-tutorial>

### Development of documentation
```console
git clone --depth 1 https://github.com/pranabdas/docusaurus-template
npm install
```

Serve locally:
```console
npm start
```

Build:
```console
npm run build
```

Deploy to GitHub pages:
```console
npm run gh-deploy
```
