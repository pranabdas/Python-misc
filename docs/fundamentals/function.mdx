Defining a function:
```python
def greeting():
    name = "Pranab Das"
    print("Good morning " + name)
```

Later calling the function to print the greeting:
<Tabs
  defaultValue="input"
  values={[
    { label: 'Input', value: 'input', },
    { label: 'Output', value: 'output', },
  ]
}>

<TabItem value="input">

```python
greeting()
```

</TabItem>

<TabItem value="output">

```python
Good morning Pranab Das
```

</TabItem>
</Tabs>
