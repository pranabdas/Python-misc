### List
It is a good idea to name lists as plural as they contain multiple elements. Like, `names`, `items` etc.
<Tabs
  defaultValue="input"
  values={[
    { label: 'Input', value: 'input', },
    { label: 'Output', value: 'output', },
  ]
}>

<TabItem value="input">

```python
names = ['Rupal', 'Pranab', 'Bhanu']
print(names)
```

</TabItem>

<TabItem value="output">

```python
['Rupal', 'Pranab', 'Bhanu']
```

</TabItem>
</Tabs>

### Tuples
Tuples are just like *lists* except that they are immutable. However, we can
update the whole tuple instead of individual entries. Here we use parenthesis
instead of square bracket to define tuples.
<Tabs
  defaultValue="input"
  values={[
    { label: 'Input', value: 'input', },
    { label: 'Output', value: 'output', },
  ]
}>

<TabItem value="input">

```python
tup = (1, 2, 3)
tup[0]
```

</TabItem>

<TabItem value="output">

```python
1
```

</TabItem>
</Tabs>

Reassigning values would result in errors
<Tabs
  defaultValue="input"
  values={[
    { label: 'Input', value: 'input', },
    { label: 'Output', value: 'output', },
  ]
}>

<TabItem value="input">

```python

```

</TabItem>

<TabItem value="output">

```python

</TabItem>
</Tabs>

<Tabs
  defaultValue="input"
  values={[

```

</Tabs>
